# upload-labs
一个帮你总结所有类型的上传漏洞的靶场

![](doc/mind-map.png)

## 运行环境

```
操作系统：推荐windows（除了Pass-19必须在linux下，其余Pass都可以在windows上运行）
php版本：推荐5.2.17(其他版本可能会导致部分Pass无法突破)
php组件：php_gd2,php_exif（部分Pass需要开启这两个组件）
apache：以moudel方式连接
```

PS：推荐使用phpstudy在windows下搭建靶机。
